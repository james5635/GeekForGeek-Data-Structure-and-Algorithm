# Greedy Algorithms

This folder contains Python implementations of classic Greedy algorithms from [GeeksforGeeks](https://www.geeksforgeeks.org/dsa/greedy-algorithms/).

## Algorithms

| Algorithm | File | Source |
|-----------|------|--------|
| Fractional Knapsack | `fractional_knapsack.py` | [GFG](https://www.geeksforgeeks.org/dsa/fractional-knapsack-problem/) |
| Dijkstra's Shortest Path | `dijkstras_algorithm.py` | [GFG](https://www.geeksforgeeks.org/dsa/dijkstras-shortest-path-algorithm-greedy-algo-7/) |
| Kruskal's MST | `kruskals_mst.py` | [GFG](https://www.geeksforgeeks.org/dsa/kruskals-minimum-spanning-tree-algorithm-greedy-algo-2/) |
| Huffman Coding | `huffman_coding.py` | [GFG](https://www.geeksforgeeks.org/dsa/huffman-coding-greedy-algo-3/) |
| Prim's MST | `prims_mst.py` | [GFG](https://www.geeksforgeeks.org/dsa/prims-minimum-spanning-tree-mst-greedy-algo-5/) |

## Quick Start

Run any algorithm directly:

```bash
python fractional_knapsack.py
python dijkstras_algorithm.py
python kruskals_mst.py
python huffman_coding.py
python prims_mst.py
```
