# Search Character in String

Search for a character in a string and return its index.

## Problem Statement

Given a string and a character, find the first occurrence of the character.

## Approaches

- **Linear search**: Iterate through string (O(n) time, O(1) space)
- **find() method**: Built-in string method (O(n) time, O(1) space)
- **Find all**: Return all indices where character appears
