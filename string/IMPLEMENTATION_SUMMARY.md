# String Problems Implementation Summary

This directory contains Python implementations for 22 Medium-level string problems from GeeksforGeeks.

## Files Created (22 implementations)

### From GeeksforGeeks URLs (15 successfully fetched)
1. **first_repeated_char.py** - Find first repeated character using frequency counting
2. **first_non_repeating_char.py** - Find first non-repeating character
3. **check_string_rotations.py** - Check if strings are rotations using KMP algorithm
4. **kth_non_repeating_char.py** - Find kth non-repeating character
5. **atoi_implementation.py** - Custom atoi() implementation with overflow handling
6. **add_binary_strings.py** - Add n binary strings
7. **check_isomorphic.py** - Check if two strings are isomorphic
8. **check_interleaving.py** - Check if string is interleaving of two others
9. **string_permutations.py** - Generate all permutations using backtracking
10. **longest_palindromic_substring.py** - Find longest palindromic substring
11. **anagram_substring_search.py** - Find all anagrams of pattern in text
12. **lexicographically_next_string.py** - Find next lexicographic string
13. **word_break.py** - Check if string can be segmented into dictionary words
14. **min_swaps_bracket_balancing.py** - Minimum adjacent swaps for bracket balancing
15. **validate_ip_address.py** - Validate IPv4/IPv6 address

### Custom Implementations (7 URLs returned 404)
16. **multiply_strings.py** - Multiply two large numbers as strings
17. **remove_adjacent_duplicates.py** - Remove all adjacent duplicates using stack
18. **roman_to_integer.py** - Convert Roman numeral to integer
19. **binary_strings_no_consecutive_ones.py** - Count binary strings without consecutive 1s
20. **split_into_four_strings.py** - Split string into four distinct strings
21. **mobile_numeric_keypad.py** - Convert sentence to mobile keypad sequence
22. **shortest_path_print_string.py** - Calculate shortest path to type string on keyboard

## Structure of Each File

Each file contains:
- Problem description and approach explanation
- Main algorithm function with docstrings
- Helper functions if needed
- Comprehensive test cases with expected outputs
- Main execution block demonstrating the algorithm

## Running the Files

To run any implementation:
```bash
python3 string/<filename>.py
```

Example:
```bash
python3 string/first_repeated_char.py
python3 string/word_break.py
```

## Time and Space Complexities

All implementations include time and space complexity analysis:
- Most optimal approaches: O(n) time, O(1) or O(n) space
- String matching problems: O(n²) to O(n) depending on algorithm
- Backtracking solutions: O(n × n!) time for permutations
