# Other Algorithms

This folder contains implementations of various algorithms from [GeeksForGeeks](https://www.geeksforgeeks.org/).

## Categories

### [Bitwise Algorithms](./bitwise/)
Algorithms that operate at the bit level for efficient computation.

### [Backtracking Algorithms](./backtracking/)
Algorithms that build candidates to solutions and abandon them when they determine the candidate cannot lead to a valid solution.

### [Divide and Conquer](./divide_and_conquer/)
Algorithms that break problems into sub-problems, solve them independently, and combine their solutions.

### [Branch and Bound](./branch_and_bound/)
Algorithm design paradigm for solving combinatorial optimization problems by systematically enumerating candidate solutions.

### [Geometric Algorithms](./geometric/)
Algorithms for solving computational geometry problems.

### [Randomized Algorithms](./randomized/)
Algorithms that employ a degree of randomness as part of their logic.

## Source

All algorithms are based on tutorials from [GeeksForGeeks DSA Tutorial](https://www.geeksforgeeks.org/dsa/dsa-tutorial-learn-data-structures-and-algorithms/).
