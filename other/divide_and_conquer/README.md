# Divide and Conquer Algorithms

Collection of divide and conquer algorithm implementations in Python. Divide and conquer is an algorithm design paradigm that works by recursively breaking down a problem into two or more sub-problems of the same or related type, until these become simple enough to be solved directly.

| File | Description |
|------|-------------|
| `allocate_minimum_pages.py` | Allocate minimum number of pages |
| `binary_search.py` | Binary Search |
| `convex_hull_divide_conquer.py` | Convex Hull using Divide and Conquer |
| `frequency_limited_range.py` | Frequency in limited range array |
| `inversion_count.py` | Inversion count using merge sort |
| `max_min_array.py` | Maximum and minimum in array |
| `merge_sort.py` | Merge Sort |
| `modular_exponentiation.py` | Modular exponentiation |
| `power_xn.py` | Calculate pow(x, n) |
| `quick_sort.py` | Quick Sort |
| `quickhull.py` | QuickHull for Convex Hull |
| `search_sorted_2d_array.py` | Search in sorted 2D array |
| `skyline_problem.py` | Skyline problem |
| `square_root_integer.py` | Integer square root |
| `strassen_matrix_multiplication.py` | Strassen's Matrix Multiplication |
| `tiling_problem.py` | Tiling problem |
