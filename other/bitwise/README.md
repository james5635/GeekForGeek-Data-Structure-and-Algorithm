# Bitwise Algorithms

Collection of bitwise algorithm implementations in Python. These algorithms demonstrate various bit manipulation techniques used in competitive programming and technical interviews.

| File | Description |
|------|-------------|
| `add_bit_strings.py` | Add two bit strings |
| `binary_representation.py` | Binary representation of a given number |
| `bitmasking_tsp.py` | Bitmasking DP for Traveling Salesman Problem |
| `booth_multiplication.py` | Booth's Multiplication Algorithm |
| `check_binary_palindrome.py` | Check if binary representation is palindrome |
| `check_bleak_number.py` | Check if a number is bleak |
| `check_equal_without_operators.py` | Check if two numbers are equal without operators |
| `check_integer_overflow.py` | Check for integer overflow |
| `check_kth_bit.py` | Check whether k-th bit is set or not |
| `check_sparse_number.py` | Check if a number is sparse |
| `closest_numbers_same_set_bits.py` | Closest numbers with same number of set bits |
| `compute_parity_xor_table.py` | Compute parity using XOR table lookup |
| `copy_set_bits_in_range.py` | Copy set bits in a range |
| `count_pairs_common_digit.py` | Count pairs with at least one common digit |
| `count_set_bits.py` | Count set bits (Brian Kernighan's algorithm) |
| `detect_opposite_signs.py` | Detect if two integers have opposite signs |
| `euclidean_with_bitwise.py` | Euclid's algorithm using bitwise operations |
| `find_odd_occurring.py` | Find number occurring odd number of times |
| `find_parity.py` | Find parity of a number |
| `find_position_only_set_bit.py` | Find position of the only set bit |
| `find_significant_set_bit.py` | Find significant set bit of a number |
| `find_two_non_repeating.py` | Find two non-repeating elements in array |
| `float_to_binary.py` | Convert floating point to binary |
| `generate_gray_codes.py` | Generate n-bit Gray codes |
| `gray_to_binary_conversion.py` | Gray to Binary and Binary to Gray conversion |
| `is_power_of_2.py` | Check if number is power of 2 |
| `karatsuba_multiplication.py` | Karatsuba fast multiplication |
| `longest_sequence_1s_with_flip.py` | Longest sequence of 1s with one flip allowed |
| `maximum_subarray_xor.py` | Find maximum subarray XOR |
| `min_max_without_branching.py` | Min/max without branching |
| `modulus_by_power_of_2.py` | Modulus division by power of 2 |
| `modulo_2_binary_division.py` | Modulo-2 binary division (CRC) |
| `next_higher_same_set_bits.py` | Next higher number with same set bits |
| `nth_binary_palindrome.py` | N-th binary palindrome number |
| `pandigital_concatenation.py` | Pairs with pandigital concatenation |
| `position_rightmost_set_bit.py` | Position of rightmost set bit |
| `rotate_bits.py` | Rotate bits of an integer |
| `russian_peasant_multiplication.py` | Russian Peasant multiplication |
| `set_kth_bit.py` | Set k-th bit in a given number |
| `smallest_of_three_without_comparison.py` | Smallest of three without comparison |
| `smallest_power_of_2_ge.py` | Smallest power of 2 greater than or equal to n |
| `square_without_operators.py` | Calculate square without multiplication operators |
| `swap_bits.py` | Swap bits in a given number |
| `swap_without_temp.py` | Swap without temporary variable |
| `turn_off_rightmost_set_bit.py` | Turn off the rightmost set bit |
| `xor_encryption.py` | XOR encryption with shifting |
| `xor_without_xor.py` | XOR without using XOR operator |
