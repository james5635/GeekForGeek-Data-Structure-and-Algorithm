# Branch and Bound Algorithms

Collection of branch and bound algorithm implementations in Python. Branch and bound is an algorithm design paradigm for discrete and combinatorial optimization problems. It systematically enumerates candidate solutions by means of state space search, using bounds to prune branches that cannot lead to optimal solutions.

| File | Description |
|------|-------------|
| `binary_strings_branch_bound.py` | Generate binary strings using Branch and Bound |
| `eight_puzzle_branch_bound.py` | 8 Puzzle using Branch and Bound |
| `job_assignment_branch_bound.py` | Job Assignment using Branch and Bound |
| `knapsack_branch_bound.py` | 0/1 Knapsack using Branch and Bound |
| `knapsack_branch_bound_implementation.py` | 0/1 Knapsack (heap-based) |
| `knapsack_least_count_branch_bound.py` | 0/1 Knapsack using Least Count |
| `n_queen_branch_bound.py` | N Queen using Branch and Bound |
| `tsp_branch_bound.py` | TSP using Branch and Bound |
| `tsp_reduced_matrix.py` | TSP using Reduced Matrix method |
