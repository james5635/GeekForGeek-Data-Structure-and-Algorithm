## How to Run

Each Python file can be run independently:

```bash
python3 recursion/<filename>.py
```

For example:

```bash
python3 recursion/factorial.py
python3 recursion/tower_of_hanoi.py
python3 recursion/n_queens.py
```

Each file includes:
- Algorithm implementation with recursion
- Base case and recursive case clearly defined
- Time and space complexity analysis
- Test cases demonstrating the algorithm
